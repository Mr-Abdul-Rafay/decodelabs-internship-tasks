# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project is a simple rule-based AI chatbot built using Python.
It responds to user inputs based on predefined rules and demonstrates the fundamentals of artificial intelligence logic.

---

## 🎯 Objectives

* Understand control flow and decision-making
* Implement a continuous chatbot loop
* Handle user input and exit conditions
* Build a basic AI system using rule-based logic

---

## ⚙️ Features

* Handles greetings like "hello", "hi", "hey"
* Responds to common questions
* Supports multiple inputs per intent
* Continuous interaction loop
* Input sanitization (lowercase + strip)
* Exit command support
* Fallback response for unknown inputs

---

## 🛠️ Technologies Used

* Python
* VS Code

---

## ▶️ How to Run

1. Open terminal in project folder
2. Run:

   ```bash
   python chatbot.py
   ```
